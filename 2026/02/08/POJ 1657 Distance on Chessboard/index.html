POJ 1657 Distance on Chessboard

发表于2026-02-08|更新于2026-02-08|POJ
|阅读量:

题目分析

  • 描述

    国际象棋的棋盘是黑白相间的8 * 8的方格，棋子放在格子中间。

    王、后、车、象的走子规则如下：

    王：横、直、斜都可以走，但每步限走一格。

    后：横、直、斜都可以走，每步格数不受限制。

    车：横、竖均可以走，不能斜走，格数不限。

    象：只能斜走，格数不限。

    写一个程序，给定起始位置和目标位置，计算王、后、车、象从起始位置走到目标位置所需的最少步数。

  • 输入

    第一行是测试数据的组数t（0 <= t <= 20）。以下每行是一组测试数据，每组包括棋盘上的两个位置，第一个是起始位置，第二个是目标位置。位置用”字母-数字”的形式表示，字母从”a”到”h”，数字从”1”到”8”。

  • 输出

    对输入的每组测试数据，输出王、后、车、象所需的最少步数。如果无法到达,就输出”Inf”.

  • 样例输入

    2

    a1 c3

    f5 f8

  • 样例输出

    2 1 2 1

    3 1 1 Inf

算法思想

  1. 处理输入：难点在于处理起始位置和目标位置，有几个需要注意的点：

    （1）需要分别计算横轴、纵轴的位移的绝对值（这里用上abs函数）；

    （2）输入的是字符串，要根据ASCII将字符串转化成整型，再根据其数值计算位移；

    //读入起始位置和目标位置
string x1,x2;
cin>>x1>>x2;
int s1 = x1[0] - 'a' + 1;
int s2 = x1[1] - '0';
int e1 = x2[0] - 'a' + 1;
int e2 = x2[1] - '0';

//计算位移
int dx = abs(s1-e1);
int dy = abs(s2-e2);

  2. 计算步长

    （1）王：横、直、斜都可以走，但每步限走一格。取横轴纵轴位移的最大值（先斜着走然后补齐横轴/纵轴所差的距离）；

    （2）后：横、直、斜都可以走，每步格数不受限制。分为走一步、走两步的情况，若是只要在横轴/纵轴上移动or斜着走能直达， 走一步，反之要走两步；

    （3）车：横、竖均可以走，不能斜走，格数不限。 若是只要在横轴/纵轴上移动能直达，走一步，反之要走两步；

    （4）象：只能斜走，格数不限。象是唯一一个可能无法到达的，故在判断象能否到达的同时可以直接进行输出。象能到达的情况只有起始位置和目标位置同色。

    那么怎么判断同色、同色情况下的步长？

    若横轴位移、纵轴位移同奇偶（相加为偶数），则同色；同色情况下，在对角线上只要一步，反之要走两步。
